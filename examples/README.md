# Examples

These files are **not** part of the ChatGPT Project runtime. They are GitHub-facing examples that help a reviewer understand and test the package.

## Sample data

- `sample-data/synthetic_artifacts.csv` — five synthetic AI/software artifacts used to demonstrate classification, reliance screening, and routing.

## Sample prompts

- `sample-prompts/sample_working_session_prompts.md` — starter prompts for batch classification, single-artifact review, and governance cleanup.

## Sample outputs

- `sample-outputs/prototype_commons_review_pack.md` — example executive review pack created from the synthetic artifact set.
- `sample-outputs/generated_artifact_classification.md` — example starter classification output.
- `sample-outputs/tool_generated_classification.md` — output generated by the local CSV classifier test.

## Design note

The examples stay outside `chatgpt-project/` on purpose. A ChatGPT Project should receive only the runtime files. GitHub receives the full repository so visitors can see the examples, test files, workflow diagram, and local tooling.
