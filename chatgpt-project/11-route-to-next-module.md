# Route To Next Module

Use this file to avoid overlap with the rest of the portfolio system.

## Routing table

| Condition | Next module |
|---|---|
| Artifact is actually a future idea | AI Opportunity Intelligence Review System |
| Artifact needs investment justification | Business Case System |
| Artifact has approved intent and needs delivery scope | Project Charter Initiation Agent |
| Multiple formalized artifacts compete for capacity/funding | Portfolio Prioritization Scoring Agent |
| Artifact needs recurring action/risk/decision follow-through | PMO Governance Operations Log |
| Artifact only needs local classification | Stay in this module |

## Handoff packet

When routing, include:

- artifact name;
- current class;
- recommended target class;
- route recommendation;
- evidence summary;
- missing evidence;
- reliance and risk summary;
- owner and backup owner;
- data/systems/permissions flags;
- required human decision;
- suggested next module.
