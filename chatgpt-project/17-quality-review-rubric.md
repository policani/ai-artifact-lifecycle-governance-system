# Quality Review Rubric

Score each output 1–5.

| Criterion | 1 | 3 | 5 |
|---|---|---|---|
| Evidence-bound | Invents facts | Mix of evidence and assumptions | Clearly separates knowns, unknowns, and assumptions |
| Lightweight | Bureaucratic | Some useful structure | Minimum process for the decision |
| Classification clarity | Vague | Class suggested | Class and rationale are clear |
| Route clarity | No next step | Generic next step | Precise route and handoff |
| Human accountability | AI appears to decide | Mixed | Human decision rights explicit |
| Module fit | Duplicates other modules | Some overlap | Clean handoff with no duplication |
| Executive usefulness | Too detailed or too shallow | Understandable | Decision-ready and concise |

## Output standard

An acceptable output should help a leader answer:

- What is this artifact?
- Who uses it?
- What does it touch?
- Is anyone relying on it?
- What class is it?
- What should happen next?
- Who must decide?
