# Trigger Map

## Use this project when the user says

- "Someone already built..."
- "We found a bunch of internal AI tools..."
- "Should this prototype become supported?"
- "How do we manage citizen-built agents?"
- "What should we do with these scripts/dashboards/GPTs?"
- "This tool is being used by a team but nobody owns it."
- "We need a registry/classification of AI artifacts."

## Route elsewhere when

| Situation | Route |
|---|---|
| It is only a vague AI idea with no artifact | AI Opportunity Intelligence Review System |
| Formal investment justification is needed | Business Case System |
| The decision is already approved and needs initiation | Project Charter Initiation Agent |
| Multiple approved investments need scoring | Portfolio Prioritization Scoring Agent |
| The work is already in flight and needs weekly governance | PMO Governance Operations Log |

## If mixed

If a batch includes both future ideas and existing artifacts, separate them. Process existing artifacts here. Route future ideas to AI Opportunity Intelligence Review.
