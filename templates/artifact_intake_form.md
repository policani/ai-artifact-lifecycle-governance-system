# Artifact Intake Record

Use this record for each artifact. Mark unknowns clearly.

| Field | Capture |
|---|---|
| Artifact name | Name or short label |
| Artifact type | Script, dashboard, GPT, agent, automation, workflow, app, integration, other |
| Builder / owner | Person or team that created it |
| Backup owner | Known backup or none |
| Users | One person, small team, department, enterprise, customers |
| Problem solved | Plain-language workflow problem |
| Current use | Experimental, occasional, recurring, operationally relied upon |
| Data touched | None, public, internal, confidential, regulated, customer, unknown |
| Systems touched | Read-only systems, write systems, system of record, external platforms, unknown |
| Decisions/actions performed | Advises, drafts, summarizes, triggers workflow, writes data, contacts users, unknown |
| Current evidence | Usage, time saved, quality improvement, revenue, cost, risk reduction, anecdote, unknown |
| Known failure mode | What breaks or degrades if it fails |
| Current documentation | None, notes, README, SOP, support doc, unknown |
| Requested outcome | Keep, share, scale, support, replace, retire, unsure |
